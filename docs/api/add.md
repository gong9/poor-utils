# add

两数相加，处理了js浮点数精度问题

## 示例

```ts
import { add } from 'poor-utils-pro'

add(0.1, 0.2)

// 0.3
```

## 参数

### 入参

| Property | Type                        | Default | Description                     |
| -------- | --------------------------- | ------- | ------------------------------- |
| arg1 | number |-|  -                           |
| arg2 | number |-| - |

### 返回值

number
