# poor-utils

## 🚀 Features
- 🎪 hope includes lodash all api also add some more better ability

## 🦄 Usage

```ts
import { add, filterEmptyVal, getUrlParams } from 'poor-utils-pro'

// example

add(0.1, 0.2) // 0.3

filterEmptyVal(
  {
    a: null,
    b: {
      c: undefined,
      d: 1,
    },
  }) // {b:{d:1}}

getUrlParams('https://www.baidu.com?a===b&c=d') // { a: "==b", c: "d" }
```

## 📦 Install
> pnpm add poor-utils-pro
